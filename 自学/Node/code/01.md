 ## Buffer对象是Node处理二进制数据的一个接口，它是Node原生提供的全局对象，可以直接使用，不需要require('buffer')

 - 实例化
  + Buffer.from(arry)
  + Buffer.from(string)
  + Buffer.alloc(size)
- 功能方法
  + Buffer.isEncoding() 判断是否支持该编码
  + Buffer.isBuffer() 判断是否为Buffer
  + Buffer.byteLength()	返回指定编码的字节长度，默认为utf8
  + Buffer.concat()	将一组Buffer对象合并为一个Buffer对象
- 实例方法
  + write()
  + slice()
  + toString()
  + toJson()
